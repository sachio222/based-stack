# Sample Analysis — Moves in Published Work

Short excerpts with commentary on what each move is doing. Not for
quotation — for pattern study.

## Move: setup-and-deflate

From the Terminal Forensics essay (in-vault):

> You type `ls`. It comes back. `Documents`. `Downloads`. Maybe
> `scripts/` if you're feeling fancy. You nod, as if the filesystem
> has just confirmed something meaningful. It hasn't.

**What's happening:**
- Short sentences stack up the small moments.
- "Feeling fancy" is the light self-aware humor beat.
- "You nod, as if…" names the reader's performance — without
  shaming them, because the writer is on their side.
- "It hasn't." is a two-word paragraph of its own, doing the
  deflate.

## Move: contrasting title with dash

"A coder's guide to colors — stop using RGB and use this instead"

**What's happening:**
- The dash sets up a rejection + promise.
- "Coder's guide" frames the audience.
- "Stop using X and use this instead" is a common pattern — strong
  verb, named target, implied payoff.

## Move: parenthetical softener in an ambitious title

"Words We'll Never Speak: AI-Native Concepts That People Can't
Use (Yet)"

**What's happening:**
- The main claim ("concepts people can't use") is strong and
  speculative.
- "(Yet)" pulls it back one notch — "I'm not saying never, I'm
  saying not currently."
- Keeps the title provocative without making it a lie.

## Move: technical precision with casual voice

> Ignore the `free` column. Linux uses idle RAM for disk cache —
> unused RAM is wasted RAM, as far as the kernel is concerned — so
> "free" on a healthy system is almost always a small, meaningless
> number.

**What's happening:**
- Direct imperative opens ("Ignore the `free` column.").
- Em-dash aside embeds the *why* in the middle of the sentence.
- "As far as the kernel is concerned" personifies the kernel
  without being cute about it.
- "Almost always a small, meaningless number" lands with judgment —
  not hedging, not academic.

## Move: dry one-liner landing

> Beginners panic. Veterans don't.

**What's happening:**
- Four words, two beats.
- Names a hierarchy without being preachy — the reader decides
  which side of it they're on.
- Works as a section-ender because there's no more to say.

## Move: turn the advice into a habit

> Three seconds. It catches the one mistake that defines careers
> in this work: running the right command in the wrong terminal.
> Do it every time. Eventually you won't notice you're doing it.

**What's happening:**
- Starts with the cost ("three seconds").
- Names the specific failure without dramatizing.
- "Do it every time" = instruction.
- "Eventually you won't notice you're doing it" = payoff — the
  habit becomes invisible, which is the gift.
